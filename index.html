  • Now there is a new system for attending class in the pandemic

    Now in Accord School there is a system called ZOOM for attending class. Mam/Sir teaches us by talking and also screen-sharing(available when you downloaded zoom)her/his laptop/desktop. She/he also asks questions and we children can answer through unmuting or we can type it in the chatbox available in zoom. We children can also show emojies like "okay, Happy Birthday" only if mam/sir said so.
